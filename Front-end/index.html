Sobre o trabalho

O presente trabalho foi proposto pelo professor Álvaro Lopes Rios, com o objetivo principal de avaliar e aplicar os conhecimentos adquiridos na área de Banco de Dados, promovendo a consolidação prática dos conceitos e técnicas estudados em sala de aula. Durante o período no IFAC, utilizamos diversos métodos aprendidos para desenvolver este projeto. O site apresentado tem como finalidade gerenciar os gastos mensais, proporcionando um melhor controle da renda por meio de monitoramento de despesas, poupança e investimentos.

Ferramentas utilizadas

Para organização e planejamento, utilizamos as seguintes ferramentas: - Chat GPT: para correção de bugs, planejamento de protótipos e definição dos modelos a serem aplicados. - GIT HUB: para facilitar a transferência de arquivos, além de acompanhar a colaboração e proatividade dos integrantes do grupo.

Línguagens utilizadas

As linguagens empregadas para a estruturação e desenvolvimento do site foram: - HTML: utilizada para construir a base do site, onde são inseridos todos os comandos e conteúdos. - Bootstrap: adotada para estilização do site, aproveitando as funcionalidades e componentes pré-prontos para facilitar o design.
